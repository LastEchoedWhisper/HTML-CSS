Excel Essential Training 2016 Learning Log


Topics I've learned in Excel Include(by chapters):

Chapter 1 - Getting Started with Excel

  • Using the menu system and Quick Access Toolbar
  • Using shortcut menus and other aspects of user interface. E.g. Formula and Status Bar

Chapter 2 - Entering Data

  • Entering Data and editing techniques, I.e. Auto Fill
  • Working with dates and times

Chapter 3 - Creating Formulas and Functions

  • Creating formulas and copying them into adjacent cells
  • Specific calculations like year-to-date totals and percentage-change formulas
  • Working with relative, absolute and mixed references

Chapter 4 - Formatting

  • Exploring font styles and effects
  • Working with alignment and Wrap Text
  • Numeric and special formatting, as well as formatting dates
  • Conditional formatting
  • Creating and using tables
  • Inserting shapes and other visual features

Chapter 5 - Adjusting Worksheet Layout and Data

  • Inserting and deleting rows
  • Hiding and unhiding rows and columns
  • Moving, copying, and inserting data

Chapter 6 - Printing

  • Exploring the Page Layout tab and Page Layout view
  • Working with Page Setup and printing controls

Chapter 7 - Charts

  • Creating charts and exploring chart types
  • Formatting charts(detailed)

Chapter 8 - Adjusting Worksheet Views

  • Freezing and unfreezing panes
  • Splitting screens horizontally and vertically
  • Collapsing and expanding data

Chapter 9 - Multiple Worksheets and Workbooks

  • Displaying multiple worksheets and workbooks
  • Renaming, inserting, and deleting sheets(Shift + F11)
  • Moving, copying and grouping sheets
  • Using formulas to link worksheets and workbooks

Chapter 10 - IF, VLOOKUP, and Power Functions

  • Using IF functions and relational operators
  • Vlookup, both approximate and exact data
  • CountIF

Chapter 11 - Security and Sharing

  • Unlocking cells and protecting worksheets
  • Protecting and assigning passwords to workbooks

Chapter 12 - Data Management Features

  • Sorting Data
  • Using filters
  • Splitting text into multiple columns
  • Using Validation tools

Chapter 13 - Data Analysis Tools

  • Using Goal Seek
  • Using Solver
  • Using Scenario Manager

Chapter 14 - PivotTables

  • Creating Pivot Tables
  • Manipulating PivotTable Data
  • Using slicers to clarify and manipulate fields
  • Using PivotCharts

Chapter 15 - Macros

  • Creating a macro
  • Running a macro
